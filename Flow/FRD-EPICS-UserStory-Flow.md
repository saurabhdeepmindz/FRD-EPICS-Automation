Step1: Create the PRD.
---------------------------
Refer the curent requirements shared as text or a document
Based on that 1st run a checklist (PRD-Template-Checklist) to identify the gaps
For each gap ask the customer for the required response
Save all the information
Create the PRD incrementally using the template PRD-Template
Recheck using checklist for any gaps
Once all gaps are filled,complete the PRD

Step2: Create EPICs
------------------------------
Based on the PRD create the list of EPICs using the template EPICs-List-Template
Incrementally create each EPIC using the EPICs-Template
Check the EPIC completeness using the EPICs-Template-Checklist
Create the RTM of PRD and EPICs incrementally using the PRD-EPICs-RTM-Template
